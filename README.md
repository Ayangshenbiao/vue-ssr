# vue-ssr

有ssr相较于spa的优点以外，增加了前端日志功能

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9999
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
